# dissmodel-sysdyn

System Dynamics models for [dissmodel](https://github.com/LambdaGeo/dissmodel).

## Installation

```bash
pip install dissmodel-sysdyn
```

## Usage

### CLI

```bash
python src/dissmodel_sysdyn/cli/sysdyn_sir.py
```

### Streamlit

```bash
streamlit run src/dissmodel_sysdyn/streamlit/sysdyn_all.py
```
